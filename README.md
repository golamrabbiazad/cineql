# CineQL

## Description

[graphql-yoga](https://github.com/prisma-labs/graphql-yoga) is fully-featured GraphQL Server with focus on easy setup, performance & great developer experience.

## Installation

```bash
# project setup
$ npm install
```

## Running the app

```bash
# development
$ npm run start:dev

# build project
$ npm run build

# production mode
$ npm run prod
```

play at [http://localhost:9300/](http://localhost:9300/)
